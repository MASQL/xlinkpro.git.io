1. Persiapan
Pastikan kamu menggunakan browser modern (Chrome/Safari/Edge).

2. Sisi Monitor (Laptop)
Buka aplikasi dan pilih MODE MONITOR.

Salin ID MONITOR unik yang muncul (misal: 8822-abcd).

Tunggu hingga koneksi dari HP masuk.

3. Sisi Join (HP)
Buka aplikasi dan pilih MODE JOIN.

Masukkan ID MONITOR dari Laptop tadi.

Klik START STREAMING dan berikan izin akses kamera.

Gunakan tombol SWITCH untuk ganti kamera atau FLASH untuk pencahayaan.
